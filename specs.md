# Personal Portfolio Specifications

## 1. Overview
* **Target Audience:** Potential employers, clients, and tech peers.
* **Primary Goal:** Showcase my projects, skills, and contact information in a clean, professional format.
* **Key Outcome:** Encourage visitors to consider me and reach out.
<br><br>

## 2. Tech Stack & Tools
* **Frontend:** HTML5, CSS, JavaScript
* **Hosting/Deployment:** GitHub
* **Design/Wireframing:** Figma
<br><br>

## 3. Structure & Content (Sitemap)
### Single Page Layout / Sections:
1. **Home Page**
   * Brief introduction (Name, Role, Design philosophy).
   * 3 Projects.
   * Mastered applications.
   * Contacts (Email, Linkedin, Github, Figma).
   * Copyright notice.
2. **Projects Page**
   * All projects - each clickable for details.
   * *Per Project:* Title, images, short description, tags/tech used, linkes to GitHub & live demo.
   * Copyright notice.
4. **Contact / Footer**
   * Simple contact form.
   * Direct email link.
   * Social links (GitHub, LinkedIn, Figma).
   * Copyright notice.
<br><br>

## 4. Design & Style Guide
* **Theme:** Clean, modern, minimalist.
* **Color Palette:**
  * Background: `#F8F5F0` (Light Milk-tea)
  * Main: `#E8DFD5` (Milk-tea)
  * Highlight: `#90A96E` (Matcha)
  * Stroke: `#909090` 70% (Light-Transparency Grey)
* **Typography:**
  * Headings: Loved by the King
  * Body: Loved by the King
* **Responsiveness:** Mobile-first approach (must look great on phones, tablets, and desktops).
<br><br>

## 5. Key Features & Requirements
- [ ] Responsive navigation bar with smooth scrolling to sections.
- [ ] Interactive project cards with hover effects.
- [ ] Working contact form & mail link.
- [ ] Fast load times (optimized images, minimal heavy external libraries).
<br><br>

## 6. Content Checklist
- [ ] Resume (PDF version for download)
- [ ] High-quality screenshots/GIFs for each project
- [ ] Written descriptions for each project
- [ ] Profile photo or avatar
